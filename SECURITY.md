# Security Policy

## Supported Versions

This repository is actively maintained. The current version on the 
main branch is the supported version.

| Version | Supported |
|---------|-----------|
| main    | ✅        |

## Reporting a Vulnerability

If you discover a security vulnerability in this repository, please 
report it responsibly.

**To report a vulnerability:**
- Email: susanaldridge555@icloud.com
- Please include a description of the issue, steps to reproduce, 
  and any relevant details.

**What to expect:**
- You will receive acknowledgment within 5 business days
- I will investigate and respond with next steps within 10 business days
- I will notify you when the vulnerability has been resolved

Thank you for helping keep this project secure and trustworthy.
